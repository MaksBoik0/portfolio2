ПОРТФОЛИО. ПРОЕКТ HTML. ГЕНЕРАТОР МЕМОВ.

Для того, чтобы проект работал, необходимо:
	1) Создать две папки: static и templates
	2) В папку templates закинуть файл index.html
	3) В папке создать ещё две папки: css и img
	4) В папку css пометить файл style.css
	5) В папку img поместить все картинки
	6) Вайл main.py оставить без папки.

По итогу у нас дожно выйти три главных файла (Две папки и файл main.py). В папке static есть две папки.
